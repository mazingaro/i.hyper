DESCRIPTION

The purpose of i.hyper.import is to import hyperspectral imagery into a GRASS GIS 3D raster (raster_3d). It converts spectral image bands into a 3D cube and attaches per-band metadata: wavelength, FWHM, valid, and unit. Currently only EnMAP L2A products are supported. Parameter input is a directory path pointing to an uncompressed EnMAP L2A product folder, which must contain both the SPECTRAL_IMAGE.TIF and the corresponding METADATA.XML. Parameter output defines the base name for the output raster_3d map and all temporary bands. The module parses metadata, checks pixel values, identifies unusable bands, and stores spectral metadata in the output cube.

NOTES

Bands that are fully zero, fully NULL, or contain negative values are marked with valid: 0. Others are valid: 1. GDAL stderr messages about missing statistics are suppressed to avoid misleading warnings. The resulting hyperspectral 3D raster map contains reflectance values from 0 to 1. All temporary rasters are cleaned up after conversion to raster_3d.

EXAMPLE

i.hyper.import input=/data/ENMAP_PRODUCT/ output=enmap product="EnMAP L2A"
